# Suggested Commands

Development:
- `pnpm dev`: Start development server using tsx watch and dotenv.
- `pnpm build`: Build the project into `dist`.
- `pnpm start`: Run the production build.

Code Quality:
- `pnpm typecheck`: Type check with tsc.
- `pnpm lint`: Lint code with Biome.
- `pnpm format`: Format code with Biome.
- `pnpm test`: Run tests with Vitest.

Database (Drizzle):
- `pnpm dz:generate`: Generate migrations.
- `pnpm dz:migrate`: Apply migrations.
- `pnpm dz:push`: Push changes directly to the database.
- `pnpm dz:studio`: Open Drizzle Studio.
